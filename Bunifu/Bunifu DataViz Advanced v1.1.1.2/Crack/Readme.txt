1. Unzip "Bunifu UI WinForms" to any directory
2. Add Bunifu components from this directory to toolbox
3. Create new WinForms application and drag & drop any Bunifu component onto the form
4. ActivationDialog will be displayed
4.1. Select "Use Offline verification code" in ActivationDialog
4.2. Enter any eMail-address and LicenseKey created with Activator and click "Activate"
4.3. Copy "Machine Signature Code" displayed in ActivationDialog and paste into Activator (=> RequestCode)
4.4. Generate ActivationCode with Activator
4.5. Copy ActivationCode from Activator and paste into ActivationDialog ("Offline Verification Code")
4.6. Finish ActivationDialog by clicking "Verify And Complete"
?
Note:
ActivationData are stored in
- Bunifu UI WinForms.blf and
- Bunifu UI WinForms.blf.entropy
in directory C:\ProgramData.
